# VoxelMod (VM)

A moderation bot for the [Voxel Fox Discord server](https://discord.gg/vfl).

## Commands

* [x] `/ban`
    * [x] `/tempban`
        * [x] Automatic unban.
    * [x] Bans user.
    * [x] Add report to user.
* [x] `/unban`
* [x] `/report`
    * [x] Add report to user.
* [x] `/warn`
    * [x] Add report to user.
* [x] `/mute`
    * [x] Delete Messages
    * [x] Add report to user.
* [x] `/unmute`
    * [x] Add report to user.
* [x] `/history`
    * See the punishment history for a given user.
    * warnings, mutes, bans (and durations), reports (perhaps)
* [ ] `/clear`
* [ ] `/logs`
* [ ] Message change logs
* [ ] Message delete logs
